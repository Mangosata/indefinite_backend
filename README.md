# Indefinite_backend

#### Don't get a good name now, just practice..

## 使用Springboot构建后端，实现登录，注册，查询，修改等功能。

### 本项目使用IntelliJ IDEA实现，可用以下方法实现在本地运行🍳

1. 使用git克隆到本地

   ```
   git clone https://github.com/Mangosata/indefinite_backend.git
   ```

2. 使用IDEA将项目导入

3. 等待IDEA构建Gradle

4. 运行DemoApplication.java
